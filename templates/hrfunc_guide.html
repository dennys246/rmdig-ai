Apply HRfunc to Your fNIRS Data

Achieve higher precision preprocessing of functional near infrared spectroscopy (fNIRS) data through HRF modeling and neural activity estimation. HRfunc enables researchers to generate spatially distributed hemodynamic response functions (HRF), recover convolved neural activity through signal and HRF deconvolution, enabling higher precision neural activity recovery from hemoglobin signals.

To get started, follow these steps:
  1. Install HRfunc package using pip:
    pip install hrfunc
  2. Create an hrfunc.montage object in python and pass in a fNIRS scan object to orient the tool to your fNIRS montage.
  3. Estimate a subject specific HRF estimate by calling to the montage.estimate_hrfs(scan, events)
  4. Estimate a subject-pool wide HRF estimate per channel by calling to montage.generate_distribution() and save your results for future use.
  5. (optional) If succesfully publishing a paper on the dataset used to estimate the HRF from, upload your estimated HRFs alongside your DOI (pre-print is ok) for community usage
