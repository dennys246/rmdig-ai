Open-source Tools



Apart of RMD's mission, all tools are made available to the public to validate our own work and accelerate scientific innovation through collaboration.

BOLDnet

BOLDnet is a tool for analyzing fMRI data using deep learning. It leverages BIDS structure to automatically detect data and build a custom convolutional neural network (CNN) with a unique spatial attention mechanism to learn intricate non-linear relationships in the data. A pre-trained model examining emotional valence in children's brains is available for transfer learning, however the tool will work for any categorical or regression experimental paradigm.

HRFunc

HRFunc is a python library for modeling hemodynamic response function (HRF) across channels in fNIRS data, leveraging a hybrid tree and hash table data structure to enable quick look up of spatially or contextually relavent HRFs. The Abominable SnowGAN

The SnowGAN is a generative adversarial network (GAN) for generating synthetic pictures of snow leveraging the snowpack dataset collected. The ultimate goal for this tool is to provide a pre-trained model that understands snowpack characteristics that can be transfered to other problems like avalanche risk.
